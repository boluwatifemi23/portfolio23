# Developer Portfolio

A modern, responsive portfolio website showcasing my projects, skills, and experience as a full-stack developer.

Designed to highlight real-world applications, clean UI, and performance-focused development.

## 🚀 Live Demo
https://portfolio-lemon-three-1zuqwbb036.vercel.app

## 🧰 Tech Stack
Next.js · TypeScript · Tailwind CSS · Framer Motion

## ✨ Features
- Fully responsive design across all devices
- Smooth animations and transitions
- Project showcase with live demos and GitHub links
- Contact section for direct communication
- Clean and modern UI with strong visual hierarchy


## 📁 Project Structure
- `src/` — Main application code (components, pages, logic)
- `public/` — Static assets
